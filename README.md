# Homebrew Tap for agentio

CLI for LLM agents to interact with communication and tracking services.

## Installation

```bash
brew tap plosson/agentio
brew install agentio
```

## Upgrade

```bash
brew upgrade agentio
```

## Usage

```bash
agentio --help
```
